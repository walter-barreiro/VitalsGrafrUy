Detailed Description

Vitalsgrafr UY - Proyecto Sistemas Embebidos 2020 - Fing Este módulo se encarga de procesar los datos para ser graficados y del chequeo de los valores para activación de alarma .

datos pregrabados son copiados a un array de 8 bits, guardándolos en otro array en dos etapas para luego ser borrados

para graficar los nuevos datos. Los datos son ajustados a la escala de cantidad de pixels designada. Chequea además si los valores medios de PR, IBP y SPO2 están fuera de rango para activación de alarma.
